🛠️ TROUBLESHOOTING:
════════════════════

Problem: "Python not found"
Solution: Install from https://www.python.org (check "Add to PATH")

Problem: "npm not found"
Solution: Install from https://nodejs.org

Problem: "Port 8000 already in use"
Solution: Use different port - edit backend main.py last line

Problem: "CORS errors"
Solution: Ensure backend is running at http://localhost:8000

Problem: "Module not found"
Solution: 
  Backend: pip install -r requirements.txt
  Frontend: npm install
